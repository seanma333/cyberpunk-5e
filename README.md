# Cyberpunk 5e

A Foundry VTT module for running a cyberpunk-fantasy campaign using the D&D 5e system. Keeps the magic and races of classic D&D while layering in a futuristic, dystopian world of megacorporations, cybernetics, and firearms.

Inspired by games like Shadowrun and Cyberpunk 2020, but designed to feel like a natural extension of 5e rather than a system replacement.

## What's Included

- **Firearms** — simple and martial, with custom properties (Automatic, Scatter, Reload, etc.)
- **Weapon Modifications** — permanent mods and detachable attachments with an in-game apply/detach workflow
- **Cyberpunk Armor** — reskinned 5e armor with cyberpunk flavor
- **Cyberware** — tiered augmentations using attunement slots
- **Explosives** — grenades and deployables with tiered damage
- **Gear & Tools** — modern equipment, hacking tools, vehicles, and more
- **Backgrounds** — cyberpunk-flavored alternatives to core 5e backgrounds
- **Custom Conditions** — expanded status icon set
- **Chat Card Buttons** — Reload and Apply Mod buttons directly on item chat cards

## Compatibility

- Foundry VTT: v12
- D&D 5e system: 3.3.1

## Installation

Install via manifest URL:
```
https://raw.githubusercontent.com/seanma333/cyberpunk-5e/main/module.json
```

Or clone the repo directly into your Foundry `Data/modules/` directory.
